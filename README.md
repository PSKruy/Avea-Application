# Good Weather
Open source weather app for Android.

## Features
* Different languages: Basque, Belarusian, Czech, English, French, German, Japanese, Spanish, Polish, Russian
* Current weather
* 7 day forecast
* Many locations
* Notifications
* Support different measuring units
* Ad-free
